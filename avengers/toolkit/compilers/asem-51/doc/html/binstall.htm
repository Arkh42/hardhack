II. BOOT-51 Installation

The BOOT-51 support files are usually installed on the host system together with the latest ASEM-51 version. Of course BOOT-51 must be installed on the MCS-51 target system, too.
Before burning BOOT-51 into an EPROM, it must be customized for the target board, and one or two cables must be tailored for local requirements.
If the design of the target system is not yet finished, some basic knowledge, concerning the general design of MCS-51 evaluation boards, might be helpful to meet the hardware requirements from the beginning.
