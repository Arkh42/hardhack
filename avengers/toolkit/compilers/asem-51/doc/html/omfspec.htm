The Intel OMF-51 Format

Intel has published a detailed specification of their OMF-51 object module format.
The last version 5.0 has been released in September 1982.

The original documents are provided in a very old, unknown text format.
However, I got a fair print-out with Microsoft Word 97.
Linux users can print them with Applixware Office.
If you fail to print or view the original files, you can read them online.
