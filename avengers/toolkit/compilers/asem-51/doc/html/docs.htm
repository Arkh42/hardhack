Document Navigation

Some of the above documents are sets of HTML pages, which have been derived from a previous ASCII edition. Since ASCII files can only be read sequentially, these documents still have a corresponding structure. To ease the navigation, especially sequential reading, within a hypertext edition, a set of navigation buttons has been placed at the bottom of all HTML pages:

Navigation Buttons
Home go directly to the table of contents
Up return to a higher chapter or document level
Back go back to the previous page of a document
Next step forth to the next page of a document

The Back and Next buttons can be used to step back and forth sequentially. The Up button allows to return to a higher chapter or document level. The Home button leads directly to the table of contents of a document.
