Appendix C

BOOT-51 Error Messages


Error MessageMeaning
checksum errorIntel-HEX record checksum is not correct.
illegal commandA command other than U or G has been entered.
illegal hex digitHex digit expected. Another character received.
illegal record IDIntel-HEX record type is greater than 1.
unexpected characterThe received Intel-HEX record doesn't start with ':', or doesn't end with CR, LF, or CR/LF.
